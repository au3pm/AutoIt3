AutoIt 3.0.94 Highlighter for UltraEdit (www.ultraedit.com)
from Marc René Delhalle <mrd@gmx.de>

Installation:
open the file "wordfile.txt" in your Ultraedit directory
and copy the content of "Ultraedit-hl.txt" at the bottom
of the file.

If necessary, change the parameter in the first Line
to another number (/L12 - L20). Only needed if you've
already added another language to this file.
